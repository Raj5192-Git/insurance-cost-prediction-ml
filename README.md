# Medical Insurance Cost Prediction using Machine Learning

## Project Overview
This project aims to predict medical insurance costs based on individual attributes such as age, gender, BMI, number of children, smoking status, and region. By applying machine learning regression techniques, the model helps estimate insurance charges and understand the key factors influencing medical expenses.

## Problem Statement
Medical insurance costs vary significantly across individuals due to lifestyle and demographic factors. Manually estimating insurance charges is inefficient and inaccurate. This project builds a data-driven machine learning model to predict insurance costs and identify the most impactful features.

## Dataset
- Source: Medical Insurance Dataset
- Format: CSV
- Key Features:
  - age
  - sex
  - bmi
  - children
  - smoker
  - region
  - charges (target variable)
